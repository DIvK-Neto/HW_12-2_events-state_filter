# Используемые технологии

## Основные инструменты

- **React 19** – библиотека для построения интерфейсов.
- **Vite** – быстрая сборка и сервер разработки.
- **JavaScript (ES6+)** – современный синтаксис, модули, стрелочные функции.
- **CSS** – адаптивная сетка на CSS `column-count`, медиазапросы.

## Кодстайл

- **ESLint** – проверка качества кода (flat‑конфиг, правила для React и React‑hooks).
- **Prettier** – автоматическое форматирование (единый стиль).

## Структура компонентов

- `Portfolio` – классовый компонент, хранит состояние.
- `Toolbar` – функциональный компонент без состояния.
- `ProjectList` – функциональный компонент без состояния.

## Адаптивность

- Использованы CSS-медиазапросы для изменения количества колонок в сетке проектов.
- При ширине экрана менее 768px – 2 колонки, менее 480px – 1 колонка.
