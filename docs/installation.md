# Установка и запуск

## Требования

- Node.js 20+ (LTS)
- Yarn (пакетный менеджер)

## Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/ваш-логин/название-репозитория.git
cd название-репозитория
```
2. Установите зависимости:

```bash
yarn install
```
## Запуск в режиме разработки

```bash
yarn dev
```
Сервер запустится на `http://localhost:5173/`. Откройте этот адрес в браузере, чтобы увидеть приложение.

## Сборка для продакшена

```bash
yarn build
```
Собранные файлы появятся в папке `dist`.

## Линтинг и форматирование

> **Важно:** Проверить код:
```bash
yarn lint
```
> **Важно:** Автоматически отформатировать:
```bash
yarn format
```
## Деплой на GitHub Pages

Если вы настроили GitHub Pages, после пуша в ветку `main` автоматически запустится сборка и деплой (при наличии настроенного CI/CD). Ссылка на готовое приложение:

`https://ваш-логин.github.io/название-репозитория/`
